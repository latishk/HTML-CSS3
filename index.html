What is CSS?
Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation semantics (the look and formatting) of a document written in a markup language. Wikipedia
What is CSS?
Cascading Style Sheets (CSS) is a simple mechanism for adding style (e.g., fonts, colors, spacing) to Web documents. W3
Abstract
CSS transforms allows elements styled with CSS to be transformed in two-dimensional or three-dimensional space. This specification is the convergence of the CSS 2D and 3D transforms.

transform Basics Explained
  • X-axis increases horizontally to the right.
  • Y-axis increases vertically downwards.
  • Z-axis increases towards the viewer, perpendicular to the plane of the screen.
  • Rotation degrees increase in the clockwise direction.
transform-origin
    Name: transform-origin
    Value: [ <percentage> | <length> | left | center | right | top | bottom ]
    Initial: 50% 50%

2D Transforms
Some 2D values of the transform property are given below:
translate(dx, dy) Specifies a 2D translation by horizontal distance dx and vertical distance dy.
translateX(dx) Specifies a 2D translation by horizontal distance dx.
translateY(dy) Specifies a 2D translation by vertical distance dy.
scale(sx, sy) Specifies a 2D scale operation by the [sx, sy] scaling vector. If sy is not provided, it assumes sy = sx
scaleX(sx) Specifies a 2D scale operation using the [sx, 1] scaling vector, where sx is given as the parameter.
scaleY(sy) Specifies a 2D scale operation using the [1, sy] scaling vector, where sy is given as the parameter
rotate(theta) Specifies a 2D rotation by the angle specified in the parameter about the origin of the element.
Demo: translate
a 
#a:hover {

	-webkit-transform: translate(40px, 40px);

	-moz-transform: translate(40px, 40px);

	-o-transform: translate(40px, 40px);

	transform: translate(40px, 40px);

}

3D Transforms
Some 3D values of the transform property are given below:
translateZ(dz) Specifies a 3D translation along the Z-axis by distance dz.
rotateX(theta) Specifies a 2D translation by horizontal distance dx.
rotateY(phi) Specifies a 2D translation by vertical distance dy.
rotateZ(psi) Specifies a 2D scale operation by the [sx,sy] scaling vector. If sy is not provided, it assumes sy = sx
